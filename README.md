# open-nga

此项目是 https://github.com/Justwen/NGA-CLIENT-VER-OPEN-SOURCE 的分支

## 新特性

1. 支持自动签到，默认关闭
2. 支持 NG 娘表情
3. 发表情从发图片 url 改成了 bbcode
4. 帖子跳到版面
5. 常用的表情
6. 发帖上传的图片宽度调整
7. 评分正常显示
8. 投票正常展示
9. 备注用户
